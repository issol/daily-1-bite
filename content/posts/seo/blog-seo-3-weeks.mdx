솔직히 말하면 처음엔 반신반의했어요. "SEO 최적화하면 정말 방문자가 늘어날까?" 블로그 시작한 지 6개월, 하루 방문자 수가 고작 30명 정도였거든요. 그런데 3주간의 집중 최적화 작업 후, 지금은 하루 평균 90명이 들어와요. 어떻게 했는지 타임라인별로 정리해드릴게요.

## 1주차: 구조화 데이터 추가 & 메타 태그 최적화

![웹 개발 코드 편집기에서 메타태그 작성 중인 모습](https://images.unsplash.com/photo-1551288049-bebda4e38f71)

_메타 태그를 하나하나 수정하는 작업이 시작이었습니다_

첫 주에는 기본 중의 기본부터 손봤어요.

### 1. 메타 태그 다시 쓰기

기존에는 그냥 대충 "블로그 제목 - 부제목" 이런 식으로만 써놨더라고요. 이걸 완전히 갈아엎었어요.

- **타이틀 태그**: 40-60자 사이로 맞추고, 핵심 키워드를 앞쪽에 배치
- **메타 디스크립션**: 120-158자로 작성, 클릭하고 싶게 만드는 문구 추가
- **키워드 앞배치 전략**: "방법", "후기" 같은 검색 의도 단어를 제목 앞부분에

예를 들어 이런 식이에요.

| 변경 전 | 변경 후 |
| --- | --- |
| 블로그 수익화 - 매일 한입 | 블로그 수익화 방법 3가지: 3개월 만에 월 50만원 달성한 후기 |
| AI 관련 포스팅입니다. | ChatGPT 활용법 5가지로 업무 시간 50% 단축한 실제 사례 (무료 템플릿 포함) |

### 2. 구조화 데이터(Schema Markup) 추가

이게 진짜 효과가 컸어요. JSON-LD 형식으로 Article 스키마를 모든 포스팅에 추가했어요. 구글이 내 글을 "뉴스 기사" 수준으로 인식하기 시작하더라고요.

```html
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Article",
  "headline": "포스팅 제목",
  "author": {
    "@type": "Person",
    "name": "작성자 이름"
  },
  "datePublished": "2026-01-15"
}
</script>
```

### 1주차 결과

정직하게 말씀드리면, 1주차에는 변화가 거의 없었어요. 구글 서치 콘솔에서 "처리 중"만 뜨더라고요. 참을성이 필요한 시기예요.

## 2주차: 내부 링크 구조 개선 & 이미지 최적화

![노트북에서 블로그 내부 링크 구조도 작성 중](https://images.unsplash.com/photo-1460925895917-afdab827c52f)

_내부 링크를 전략적으로 배치하는 것이 핵심이었습니다_

2주차부터 본격적으로 손이 많이 갔어요.

### 1. 내부 링크 전략적으로 깔기

기존 포스팅 50개를 전부 열어서 관련 글끼리 연결했어요.

- 한 포스팅당 최소 2-3개의 내부 링크 추가
- 앵커 텍스트에 키워드 자연스럽게 포함 (예: "이 방법"이 아니라 "ChatGPT 프롬프트 작성법")
- "시리즈" 형태로 묶을 수 있는 글들은 상단에 목차 박스 추가

이게 생각보다 효과가 좋았어요. 방문자가 한 페이지만 보고 나가는 게 아니라, 평균 2.3페이지를 보게 됐거든요. 구글이 "오, 이 블로그는 양질의 연관 콘텐츠가 많네?"라고 판단하는 것 같더라고요.

### 2. 이미지 최적화 (이게 진짜 중요)

블로그에 올린 이미지들이 용량이 너무 컸어요. 스마트폰으로 찍은 사진을 그냥 올렸거든요.

- 모든 이미지 WebP 포맷으로 변환 (TinyPNG 사이트 이용)
- 파일명을 "IMG_1234.jpg"에서 "chatgpt-prompt-example.webp"로 변경
- alt 텍스트 꼼꼼하게 작성 (검색 노출용)

### 실패한 시도도 솔직하게 공유

2주차에 "키워드 밀도 10%가 좋다"는 글을 보고 따라했다가 오히려 순위가 떨어졌어요. 구글이 키워드 스터핑으로 판단한 것 같더라고요. 자연스럽게 쓰는 게 답이에요.

### 2주차 결과

- 일 방문자: 30명 → 52명 (73% 증가)
- 페이지뷰: 35 → 120 (243% 증가)
- 평균 체류 시간: 42초 → 1분 38초

드디어 변화가 느껴지기 시작했어요!

## 3주차: 결과 분석 & 추가 최적화

![구글 애널리틱스 대시보드 트래픽 증가 그래프](https://images.pexels.com/photos/590022/pexels-photo-590022.jpeg)

_3주차에는 구글 서치 콘솔 데이터를 매일 아침 체크했어요_

### Google Search Console 실제 데이터

| 지표 | 최적화 전 | 3주 후 | 변화율 |
| --- | --- | --- | --- |
| 총 노출 수 | 1,240회 | 8,920회 | +619% |
| 총 클릭 수 | 15회 | 142회 | +847% |
| 평균 CTR | 1.2% | 3.8% | +217% |
| 평균 게재순위 | 28.4위 | 12.7위 | 15.7위 상승 |

특히 클릭률(CTR)이 1.2%에서 3.8%로 오른 게 신기했어요. 메타 디스크립션을 매력적으로 쓴 게 먹혔나봐요.

### 추가로 발견한 것들

서치 콘솔 "실적" 탭에서 "노출은 많은데 클릭이 없는 검색어"를 찾았어요. 이런 키워드들은 제목이나 설명이 별로였던 거예요. 해당 포스팅 10개를 리라이팅했더니 다음날부터 클릭이 들어오기 시작했어요.

### 3주차 최종 결과

- 일평균 방문자: 30명 → 92명 (약 3배)
- 구글 유입 비율: 23% → 67%
- 블로그 체류 시간: 42초 → 2분 14초

## 개인적으로 가장 효과가 컸던 것

3주간 작업하면서 느낀 건데, **구조화 데이터(Schema Markup)**가 생각보다 엄청 중요하더라고요. 이거 넣고 나서 구글 검색 결과에 별점, 작성일, 썸네일이 풍부하게 표시되기 시작했어요. 당연히 클릭률이 올라갈 수밖에 없죠.

그리고 의외로 **이미지 최적화**도 큰 영향을 미쳤어요. 페이지 로딩 속도가 3.2초에서 1.1초로 줄어들었거든요. 구글이 "Core Web Vitals" 점수를 높게 줬더라고요.

### 앞으로 계속할 것

- 매주 구글 서치 콘솔 체크해서 노출 많은데 클릭 없는 글 찾기
- 신규 포스팅할 때부터 메타 태그, 스키마 마크업 세팅
- 월 1회 내부 링크 점검 (깨진 링크 없는지)
