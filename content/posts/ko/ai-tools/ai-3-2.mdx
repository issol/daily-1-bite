## 시작은 ChatGPT였다
 작년 11월부터 본격적으로 AI 도구들을 써보기 시작했어요. 사실 처음엔 ChatGPT만 쓸 생각이었거든요. 근데 주변에서 Claude도 좋다더라, Gemini는 구글이라 검색이 강하다더라, Copilot은 무료라더라... 이런 얘기들 듣다 보니까 "아 그냥 다 써보자" 싶더라고요. 그래서 3개월간 ChatGPT, Claude, Gemini, Copilot 이렇게 4개를 돌아가면서 써봤어요. 유료 결제도 해보고, 무료 버전도 써보고. 지금부터 제가 실제로 느낀 점들 솔직하게 풀어볼게요.

  
## ChatGPT - 역시 1등은 1등이다
 
![ChatGPT 사용 화면 이미지](https://images.unsplash.com/photo-1686904423955-b928225c6488)

  가장 많이 쓴 도구: ChatGPT (GPT-4o) ChatGPT가 가장 편했어요. 왜냐면 대화가 자연스럽거든요. 제가 주로 쓴 용도는 이거예요:

  코딩 질문: "이 에러 왜 나는 거야?" 하면 바로 알려줘요. Python 코드 짤 때 진짜 많이 물어봤는데, 정확도가 95% 정도 되는 것 같아요. 블로그 글 초안: 키워드만 던져주면 구조 잡아주는 게 좋더라고요. 물론 그대로 쓰진 않고 제 스타일로 다시 쓰지만요. 아이디어 브레인스토밍: "이런 서비스 만들려고 하는데 어떻게 생각해?" 물어보면 진짜 같이 고민해주는 느낌이에요. 번역: 영어 문서 읽다가 막히면 바로 물어봐요. 맥락 이해하고 번역해줘서 구글번역보다 훨씬 나아요.  장점: - 대화가 자연스러워요. 마치 사람이랑 얘기하는 느낌 - 창의적인 답변 잘 줘요 - 플러그인 많아서 웹 검색, 이미지 생성 다 돼요 - 업데이트 빨라요 단점: - 가끔 거짓말해요 (환각이라고 하더라고요). 특히 최신 정보는 조심해야 돼요 - 유료 버전($20/월) 써야 GPT-4o 쓸 수 있어요 - 한국어는... 좀 어색할 때가 있어요

  
## Claude - 글쓰기 천재
 
![글쓰기 작업 화면](https://images.unsplash.com/photo-1516321318423-f06f85e504b3)

  Claude Opus 4.5를 써봤는데요, 이건 진짜 놀랐어요. 제가 20페이지짜리 PDF 보고서를 통째로 집어넣고 "이거 요약해줘" 했거든요. 그랬더니 5분 만에 핵심만 쏙쏙 뽑아주더라고요. ChatGPT는 파일 크기 제한 때문에 나눠서 넣어야 했는데 Claude는 한방에 처리됐어요. 주요 용도:

  긴 문서 분석: 200페이지짜리 논문도 읽어요. 진짜예요 한국어 글쓰기: ChatGPT보다 자연스러워요. 띄어쓰기, 조사 실수도 적고요 코드 리뷰: 제 코드 보여주면 문제점 정확하게 짚어줘요 윤리적 판단: 이건 특이한 건데, "이런 상황에서 어떻게 해야 해?" 물어보면 여러 관점에서 생각해줘요  장점: - 긴 글 처리 능력이 미쳤어요 (20만 토큰까지) - 한국어 품질 최고예요 - 거짓말 거의 안 해요 (환각 최소) - 글쓰기 톤이 자연스러워요 단점: - 속도가 좀 느려요. ChatGPT보다 답변 나오는 데 시간 걸려요 - 창의성은 ChatGPT가 더 나은 것 같아요 - 이미지 생성 안 돼요 - 역시 유료 ($20/월)

  
## Gemini - 구글의 저력
 
![구글 워크스페이스 화면](https://images.unsplash.com/photo-1573164713714-d95e436ab8d6)

  Google Gemini는 특히 리서치할 때 좋았어요. 왜냐면 구글이잖아요. 검색이 강할 수밖에 없죠. "2026년 AI 트렌드 정리해줘" 하면 최신 기사들 찾아서 정리해줘요. 실시간 인용도 달아주고요. 주요 용도:

  최신 정보 검색: "오늘 코스피 얼마야?" 이런 거 물어보면 바로 알려줘요 구글 워크스페이스 연동: Gmail, Google Docs랑 연결돼서 편해요 유튜브 영상 요약: 유튜브 링크 넣으면 내용 정리해줘요 멀티모달: 이미지랑 텍스트 동시에 처리해요  장점: - 컨텍스트 윈도우가 100만 토큰이래요 (엄청 긴 문서 처리 가능) - 구글 생태계랑 찰떡궁합 - 실시간 웹 검색 기본 제공 - 무료 버전도 괜찮아요 단점: - 대화 흐름이 좀 뚝뚝 끊기는 느낌? - 코딩은 ChatGPT나 Claude만 못해요 - 개인정보 걱정되는 사람은 구글이라 부담스러울 수도

  
## Copilot - 무료의 제왕
 
![코딩 작업 화면](https://images.unsplash.com/photo-1555066931-4365d14bab8c)

  Microsoft Copilot은 사실 기대 안 했는데 의외로 괜찮았어요. 왜냐면 무료거든요. Edge 브라우저나 Windows에서 기본으로 쓸 수 있어요. 주요 용도:

  빠른 검색: 간단한 질문은 이걸로 해결해요 Microsoft 365 연동: Word, Excel, PowerPoint에서 바로 써요 기업용 문서 작업: 회사 데이터 분석할 때 좋더라고요  장점: - 무료예요! - Microsoft 생태계 쓰는 사람한테 최적화 - 응답 속도 빨라요 단점: - 깊이가 부족해요. 복잡한 질문은 답 못 해요 - ChatGPT 기반인데 제한된 느낌 - 창의성 떨어져요

## 실망했던 점들
 
![실망한 표정](https://images.unsplash.com/photo-1556761175-4b46a572b786)

  솔직하게 말할게요. 1\. 다 똑같은 소리 한다 가끔 4개 도구에 똑같은 질문 던져봤는데요. 답변이 비슷해요. 특히 일반적인 질문은 거의 복붙 수준이더라고요. "AI가 다 거기서 거기구나" 싶었어요. 2\. 한국어는 여전히 약하다 Claude가 제일 낫긴 한데, 그래도 가끔 어색해요. 조사 틀리거나, 문맥 이상하거나. 영어권 사람들이 만든 거라 어쩔 수 없나 봐요. 3\. 최신 정보는 조심해야 함 ChatGPT가 2024년 올림픽 금메달 순위 물어봤는데 완전 개소리하더라고요. 나중에 확인해보니 틀렸어요. 최신 정보는 꼭 팩트 체크해야 돼요. 4\. 창의성 한계 "참신한 아이디어 줘" 하면 다 비슷비슷해요. 진짜 혁신적인 건 역시 사람이 내야 하나 봐요.

  
## 놀라웠던 점들
 
![놀라는 표정](https://images.unsplash.com/photo-1620712943543-bcc4688e7485)

  반대로 "와 이건 진짜 좋다" 싶었던 것들도 있어요. 1\. 코딩 속도 3배 증가 진짜예요. 예전엔 Stack Overflow 뒤지면서 한 시간 걸렸던 거 이제 10분 만에 해결돼요. 특히 에러 메시지 복붙하면 바로 해결책 알려줘서 개발 속도가 확 올랐어요. 2\. 외국어 장벽 사라짐 영어 논문, 일본어 기술 문서 이런 거 읽을 때 ChatGPT한테 물어보면 바로 번역해줘요. 맥락까지 이해해서 설명해주니까 이해가 빨라요. 3\. 24시간 멘토 새벽 3시에 코딩하다가 막혀도 물어볼 데가 있다는 게 좋아요. 사람한테 물어보기 미안한 기초적인 질문도 부담 없이 할 수 있고요. 4\. 아이디어 정리 머릿속에 뒤죽박죽인 생각들 ChatGPT한테 막 던지면 정리해줘요. "이거 구조화해줘" 하면 체계적으로 만들어주더라고요.
