좋아하는 J-Pop YouTube URL만 입력하면 AI가 가사를 분석하고 단어를 추출해서 SRS 플래시카드를 자동 생성합니다. YOASOBI, Ado, 요네즈 켄시 노래로 일본어 어휘를 재미있게 익히는 방법을 소개합니다.


## 좋아하는 노래로 일본어를 배울 수 있다면?

YOASOBI의 "アイドル"을 들을 때마다 가사가 궁금해서 번역 사이트를 뒤져본 적 있나요? 요네즈 켄시의 "Lemon"을 따라 부르면서 "이 단어가 무슨 뜻이지?" 싶었던 적은요?

저도 그랬습니다. J-Pop을 좋아하면서 일본어를 배우고 싶었는데, 방법이 너무 번거로웠습니다. 가사 사이트에서 모르는 단어를 하나하나 사전에 검색하고, Anki에 수동으로 카드를 만들고... 노래 한 곡당 30분에서 1시간이 걸렸습니다.

그래서 만든 게 [Utakoto](https://utakoto.space)입니다.


## Utakoto가 뭔가요?

한 줄 요약: **YouTube URL 하나만 입력하면, AI가 1분 내에 가사 분석 + 단어 추출 + JLPT 태깅 + 후리가나 + 번역까지 자동으로 해줍니다.**

기존에 30분 걸리던 작업이 1분이면 끝납니다.

### 작동 방식

1. **YouTube URL 입력**: 좋아하는 J-Pop 뮤직비디오의 URL을 붙여넣기
2. **AI 자동 분석**: 가사 추출, 형태소 분석(kuromoji), 단어 추출, JLPT 난이도 분류, 후리가나 생성, 한국어 번역까지 자동
3. **SRS 복습**: 추출된 단어들이 SuperMemo-2 알고리즘 기반의 플래시카드로 자동 생성. 과학적으로 검증된 간격 반복으로 장기 기억에 저장


## 다른 앱이랑 뭐가 다른가요?

일본어 학습 앱은 많습니다. 하지만 "내가 좋아하는 노래"로 배울 수 있는 앱은 거의 없습니다.

| | Utakoto | 기존 단어장 앱 |
|---|---|---|
| **노래 수** | 무제한 (YouTube URL이면 뭐든) | 고정 목록 300곡 |
| **단어 추출** | AI 자동 (1분) | 수동 입력 또는 큐레이션 |
| **JLPT 분류** | 자동 (N5~N1) | 수동 또는 없음 |
| **복습 방식** | SRS (SuperMemo-2) | 단순 반복 |
| **후리가나** | 자동 생성 | 있거나 없거나 |
| **한국어 번역** | 가사 라인별 자동 번역 | 없음 |

핵심 차이는 **"어떤 노래든"** 이라는 점입니다. 300곡 목록에 내가 좋아하는 곡이 없으면? Utakoto는 YouTube에 있는 모든 일본 노래를 지원합니다.


## 실제로 써보면?

### 1단계: 노래 추가

앱에 접속해서 YouTube URL을 붙여넣습니다. 예를 들어 YOASOBI의 "夜に駆ける(밤에 달리며)"를 추가하면, 약 1분 후에 가사와 단어가 준비됩니다.

### 2단계: 가사와 함께 학습

모바일에서는 가사가 한 줄씩 캐러셀로 표시됩니다. 각 라인마다 후리가나, 한국어 번역, 한국어 발음이 함께 보이고, 해당 라인에 포함된 단어들이 카드 형태로 바로 위에 나옵니다.

노래를 들으면서 가사를 따라가고, 모르는 단어를 바로 확인할 수 있습니다. 사전을 따로 찾을 필요가 없어요.

### 3단계: 플래시카드 복습

추출된 단어들은 자동으로 플래시카드가 됩니다. 4단계 난이도(다시/어려움/좋음/쉬움)로 평가하면, SuperMemo-2 알고리즘이 최적의 복습 간격을 계산합니다.

"좋음"으로 평가한 단어는 3일 후에, "쉬움"은 일주일 후에, "어려움"은 6분 후에 다시 나타납니다. 이 방식으로 학습하면 일반 암기보다 기억 유지율이 2-3배 높습니다.


## 3주 직접 써본 결과

솔직히 말하면, 처음에는 회의적이었습니다. "노래로 단어 외우기"는 예전에도 시도해봤는데, 결국 가사만 외우고 단어는 실전에서 못 쓰는 패턴이 반복됐거든요. 그래서 이번엔 3주 동안 의도적으로 데이터를 남기면서 써봤습니다.

**1주차: YOASOBI "夜に駆ける"로 시작**

첫 번째로 추가한 곡은 YOASOBI의 "夜に駆ける(밤에 달리며)"였습니다. 이미 멜로디는 귀에 익숙한 곡이라 가사가 더 잘 들릴 줄 알았는데, 막상 단어를 보니 N3 이상 단어가 생각보다 많았습니다.

그 주에 집중해서 외운 단어들:
- **跨ぐ(またぐ)** — 넘다, 걸쳐서 넘어가다 (N3)
- **飛び込む(とびこむ)** — 뛰어들다 (N3)
- **閉ざす(とざす)** — 닫다, 막다 (N2)
- **揺らぐ(ゆらぐ)** — 흔들리다 (N2)
- **縋る(すがる)** — 매달리다, 의지하다 (N1)

跨ぐ는 "발을 들어서 넘다"라는 구체적인 동작인데, 가사에서 "그 경계를 넘는다"는 맥락으로 나와서 이미지가 바로 각인됐습니다. 사전에서 예문으로 외울 때보다 기억이 훨씬 잘 됐습니다. 반면 縋る는 가사 맥락이 너무 감정적이고 추상적이어서 "이게 실생활에서 언제 쓰이지?"라는 의문이 계속 들었습니다. 이건 나중에 2주차에서 다시 나왔을 때야 겨우 정착됐습니다.

1주 차 말에 기억한 단어 수: 추출된 47개 중 32개를 "좋음" 이상으로 평가 완료.

**2주차: Ado "うっせぇわ"와 Fujii Kaze "Matsuri" 추가**

2주차에는 두 곡을 동시에 진행했습니다. Ado의 "うっせぇわ"는 현대 구어체가 많아서 실용적인 단어가 많았습니다. Fujii Kaze의 "Matsuri"는 반대로 아가와 가벼운 표현이 많아서 SRS 복습이 빠르게 쌓였습니다.

재미있었던 건, 두 곡에서 비슷한 단어가 겹치기 시작했다는 점입니다. **鬱陶しい(うっとうしい, 귀찮다/짜증스럽다)**는 うっせぇわ에서 변형으로 등장했고, **踊る(おどる, 춤추다)**는 Matsuri에서도 반복됐습니다. 단어들이 여러 곡에서 반복되면서 맥락이 쌓이니까 확실히 장기 기억에 남는 느낌이 달랐습니다.

이 시점에서 패턴이 보이기 시작했습니다. 일본어 가요 가사에서 유독 자주 나오는 동사군(走る, 輝く, 消える, 叫ぶ 계열)이 있고, 이것들을 곡 하나에서 외우면 다른 곡에서 자연스럽게 복습이 됩니다.

**3주차: 실력 변화 확인**

3주 차에는 JLPT N4 어휘 모의고사를 온라인에서 찾아서 풀어봤습니다. 3주 전에 같은 유형의 테스트를 풀었을 때 정답률이 61%였는데, 3주 후에는 78%로 올랐습니다. 완벽한 통제 실험은 아니지만, Utakoto로 반복 학습한 단어들이 실제로 시험 문제에서 보였고 바로 답을 고를 수 있었습니다.

**솔직한 불만 하나**: 어떤 곡들은 가사가 너무 시적이거나 고어(古語) 표현이 많아서 실용 어휘 학습에는 맞지 않습니다. 요네즈 켄시의 "Lemon"에 나오는 일부 표현들은 문어체에 가까워서, 플래시카드로 열심히 외워도 "이걸 실제 회화에서 쓸 일이 있을까?"라는 생각이 들었습니다. 반대로 Creepy Nuts의 "Bling-Bang-Bang-Born"처럼 리듬감 있는 구어체 가사는 학습 효율이 훨씬 높았습니다.

**진짜 수확**: 3주차 말부터 애니메이션을 볼 때 자막 없이도 들리는 단어가 늘어났습니다. 특히 Utakoto로 외운 단어들, 예를 들면 叫ぶ(외치다), 逃げる(도망가다), 消える(사라지다) 같은 단어들이 애니 대사에서 들릴 때 사전을 찾지 않아도 의미가 바로 왔습니다. 이게 가장 체감되는 변화였습니다.


## Anki 사용자라면 어떻게 달라지나

이미 Anki를 쓰고 있는 사람이라면 "그냥 Anki에 카드 추가하면 되는 거 아냐?"라는 생각이 드는 게 당연합니다. 저도 2년 넘게 Anki를 써온 사람으로서, 솔직하게 비교해봤습니다.

**Anki에서 잃는 것**

Anki의 가장 큰 장점은 커스터마이징입니다. 카드 템플릿을 직접 설계할 수 있고, AnkiWeb을 통해 모든 기기에서 완벽하게 동기화되며, 수천 개의 애드온으로 기능을 확장할 수 있습니다. Japanese Support 애드온을 쓰면 자동 후리가나가 들어가고, Migaku 같은 유료 애드온을 쓰면 꽤 그럴듯한 몰입 학습 환경을 만들 수 있습니다.

Utakoto로 넘어오면 이 생태계를 포기해야 합니다. 데이터 내보내기(Pro 플랜)는 가능하지만, 기존 Anki 덱과의 통합은 매끄럽지 않습니다. 그리고 Anki는 10년 넘은 검증된 SRS 엔진이지만, Utakoto는 아직 새로운 서비스입니다.

**Utakoto에서 얻는 것**

반면, 덱 제작에 드는 시간이 0입니다. Anki로 일본어 공부를 시작하려면 어떤 덱을 써야 할지, 어떤 템플릿이 좋은지 찾아보는 데만 몇 시간이 걸립니다. Utakoto는 URL 하나로 끝납니다.

그리고 결정적으로, **동기 부여의 질이 다릅니다.** Anki의 "오늘 리뷰 50개 완료"는 숫자일 뿐이지만, "좋아하는 노래 한 곡 마스터"는 감정적 만족감이 있습니다. 지속적으로 공부하는 데 이게 생각보다 큰 차이를 만듭니다.

**누가 바꿔야 하고, 누가 Anki를 유지해야 하는가**

Utakoto로 넘어오면 좋은 경우:
- 이미 J-Pop을 즐겨 듣고 있는 경우
- Anki 덱을 만드는 데 시간을 쓰기 싫은 경우
- 일본어 초중급(N5~N3)이라 기본 어휘를 음악 맥락으로 쌓고 싶은 경우
- 공부가 습관이 아직 안 된 경우 (진입 장벽이 낮음)

Anki를 유지하는 게 나은 경우:
- 이미 수천 개 단어가 쌓인 Anki 덱이 있는 경우
- 고급 학습자(N2 이상)로 JLPT 문법/독해 중심 공부가 필요한 경우
- 커스텀 카드 형식이 학습에 필수적인 경우
- 기기 간 완벽한 동기화가 필수인 경우

두 앱을 병행하는 것도 현실적인 선택입니다. Utakoto로 노래 어휘를 익히고, Anki에 그 단어들을 별도로 정리해두는 방식이 꽤 잘 맞습니다.


## 한 달 후 솔직한 평가

한 달을 써보고 나서 내린 결론을 과장 없이 정리합니다.

**잘 된 것**

SRS 타이밍이 체감상 잘 맞았습니다. "어려움"으로 표시한 단어는 짧은 간격으로 반복해서 나오고, "쉬움"으로 표시한 단어는 생각날 만한 타이밍에 다시 나왔습니다. Anki의 SM-2 구현과 사실상 동일한 알고리즘이지만, 음악 맥락이 붙어있으니까 "아, 그 노래에서 나온 단어"라는 기억 연결고리가 하나 더 생깁니다.

모바일 UX도 꽤 잘 만들어져 있습니다. 출퇴근길에 가사 한 라인씩 넘기면서 복습하는 게 일상이 됐습니다.

**안 된 것**

고어나 시적 표현이 많은 노래는 학습 효율이 떨어집니다. 클래식한 J-Pop이나 엔카 계열을 주로 듣는 분이라면, 추출된 단어들이 현대 일상 회화와 괴리가 생길 수 있습니다. 서비스 측에서 "이 노래는 구어체 비율 X%, 시어 비율 Y%"라는 정보를 미리 보여주면 노래 선택이 훨씬 쉬울 것 같습니다.

오프라인 지원이 없습니다. 지하철 터널처럼 연결이 끊기는 환경에서는 복습이 안 됩니다. 오프라인 캐싱이 지원된다면 모바일 환경에서 훨씬 쓰기 좋을 겁니다.

**이런 분께 딱 맞습니다**

J-Pop을 좋아하면서 "언제 일본어 공부하나"라고 고민하던 분에게 이 앱은 정말 잘 맞습니다. 어차피 듣는 노래에서 단어를 뽑아내니 추가로 투자해야 할 시간이 거의 없습니다. 일본어 초중급 학습자가 어휘를 늘리는 진입점으로, 또는 중급 정체기에 학습 동기를 다시 불태우는 용도로 추천합니다.

**이런 분께는 맞지 않을 수 있습니다**

N1·N2 준비 중인 고급 학습자라면 가요 어휘만으로는 시험 어휘 커버리지가 부족합니다. 독해 지문에 나오는 비즈니스·신문 어휘를 커버하는 별도 소재가 필요합니다. Utakoto는 어휘 학습의 전부가 아니라, 즐겁게 꾸준히 할 수 있는 좋은 채널 하나라고 생각하면 적당합니다.


## JLPT 학습에도 도움이 되나요?

네. 모든 단어가 JLPT N5(가장 쉬움)에서 N1(가장 어려움)까지 자동 분류됩니다.

예를 들어 YOASOBI의 "アイドル"에는:
- N5 단어 15개 (食べる, 大きい 같은 기초 단어)
- N4 단어 20개
- N3 단어 35개
- N2 단어 25개
- N1 단어 10개

이렇게 분포가 나오면 "이 노래는 N3 수준이구나"라고 바로 판단할 수 있고, 내 레벨에 맞는 노래를 골라서 학습할 수 있습니다.
